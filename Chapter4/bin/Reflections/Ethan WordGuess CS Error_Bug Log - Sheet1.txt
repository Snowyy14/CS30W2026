Error Log

Error 1:
What error message did you encounter?
FileNotFoundException: words.txt (The system cannot find the file specified)

What unexpected behavior did your program exhibit?
The program crashed right at the start before the game even began. It couldn't find the words file.

What caused the issue?
The words.txt file wasn't in the folder that Eclipse was looking in. Eclipse runs programs from the project root directory, not from the src folder where I had the file at first.

How did you fix the issue?
I moved words.txt to the project root folder (Chapter4) so that Eclipse could find it. I also added a catch block for FileNotFoundException that prints a helpful message instead of just crashing.

Error 2:
What error message did you encounter?
NumberFormatException when trying to parse the first line of the file

What unexpected behavior did your program exhibit?
The program crashed when trying to read how many words are in the file. It was reading the wrong thing.

What caused the issue?
I had a blank line at the top of my words.txt file which meant the first readLine() was getting an empty string instead of the number. Integer.parseInt("") doesn't work.

How did you fix the issue?
I added .trim() to the readLine() call to get rid of any extra whitespace, and I also made sure my words.txt file starts with the number on the very first line with no blank lines before it.
