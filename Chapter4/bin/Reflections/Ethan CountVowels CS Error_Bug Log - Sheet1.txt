No error, but the vowel count was wrong.

The program was counting fewer vowels than expected. I had a word like "APPLE" and it wasn't counting the uppercase A.

What caused the issue?
Logic error - I forgot to convert the line to lowercase before checking for vowels. Since I was only checking for lowercase a, e, i, o, u, any uppercase vowels got missed.

Added lowercaseText = lineInFile.toLowerCase() before the character loop so everything gets converted to lowercase first. After that the count was correct.
